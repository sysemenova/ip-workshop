Make sure to change the IP address in two places!

To run this and see it in action, do the following:
  - Preferably, open up two terminals
  - Run python receive.py in one of them
  - Run python send.py in the other, and watch magic happen!
You may have to substitute python with python3. Try both - one should work if you have python installed.

On Windows, you may have a firewall warning. Thanks, windows, but we'll ignore you for now!
